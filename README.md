E-Commerce Sales Dashboard – Power BI

📌Project Overview
This project is part of my internship, Task 1: Analyze e-commerce data to identify best-selling products, sales trends, and high-revenue categories. Using Power BI, I transformed raw e-commerce sales data into an interactive dashboard that provides clear and actionable business insights.

The dashboard enables users to:
1.Track total sales quantity and revenue over time.
2.Identify best-selling products and high-revenue categories.
3.Monitor year-over-year sales trends.
4.Compare country-level performance.

🎯 Objectives
1.Analyze historical e-commerce sales data.
2.Highlight top-performing products and bulk orders.
3.Compare sales performance across years (2009–2010 vs 2010–2011).
4.Visualize insights using Power BI interactive features.

🛠 Tools & Technologies
Power BI – Data visualization and dashboard creation.
Excel/CSV Dataset – E-commerce sales data.
Data Cleaning – Applied filters, data formatting, and aggregation.
